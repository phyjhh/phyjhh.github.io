# Phyj Notes & Blog

这是我的个人站点仓库，基于 Astro 构建，包含博客、笔记与发布内容。

## 目录结构
- `src/content/posts`：博客文章
- `src/content/notes`：学习笔记
- `src/content/publications`：发布/论文
- `public/`：静态资源

## 本地运行
```bash
pnpm install
pnpm dev
```

## 部署
通过 GitHub Actions 自动构建与部署。
